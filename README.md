🛰️ ApexWeather | Enterprise Intelligence EngineApexWeather is a high-performance, single-file Python weather suite built for deep atmospheric analysis. Unlike standard weather apps, ApexWeather utilizes a proprietary Vector Canvas Engine to render real-time weather data and interactive trends without relying on external graphing libraries like Matplotlib.🚀 Key Features📊 Interactive Trend Analysis: A custom-built graphing engine using Tkinter Canvas math. Features real-time hover tooltips and dynamic scaling.🎨 Vector Icon Engine: Weather icons are rendered via pure mathematical coordinates (no static .png or .jpg files used), ensuring infinite scalability and low memory overhead.🍃 Dual-API Integration: Synchronizes data from both the Open-Meteo Forecast API and the Air Quality Index (AQI) API for a 360-degree environmental view.💾 Persistent SQLite3 Storage: Features a built-in database layer to save user preferences, theme states, and localized search history.🧵 Multi-Threaded Sync: Network requests are offloaded to background threads to ensure a butter-smooth "Extraordinary UI" experience without frame drops.📅 7-Day Atmospheric Forecast: Detailed long-range projections including temperature variance and WMO weather condition mapping.🛠️ Tech StackLanguage: Python 3.9+UI Framework: CustomTkinter (Modernized Tkinter wrapper)Data Format: JSON (RESTful API interaction)Database: SQLite3Graphics: Native Tkinter Canvas (Vector Math)Networking: requests (Asynchronous handling)📦 InstallationTo deploy the ApexWeather environment, ensure you have Python installed, then follow these steps:Clone the repository:Bashgit clone https://github.com/yourusername/apex-weather-pro.git
